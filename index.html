Unofficial Rap Genius App

for android

This is the unofficial Rap Genius android app.
It parses explanations and lyrics from RapGenius.com.
Officially supports English, other languages may work.
Supports 2.3+.

DISCLAIMER:

This app's creation is in no way associated with Rap Genius.

Installation Instructions


Two choices for installation:
  1. Easiest way to install the app is to just grab the .apk and install it onto your device (uncheck the option that blocks installing apks in the settings).
  2. Download the entire source and import it into Eclipse*, then:
    • Build/Compile/Run it on your device,
    • Export it to an .apk yourself (File -> Export..).
      • Exporting would require generating a key store.
  3. Try and import the project to Android Studio, and build it that way. You'll have to modify the build.gradle however.
*Be sure to remove the exclude screenshots from Eclipse after downloading.

If you had installed this app from the market, you should probably remove that version before installing this to avoid any problems.
